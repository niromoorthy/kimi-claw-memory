# HEARTBEAT.md

Daily checks to run during heartbeat (2-4x per day):

## Checklist

- [x] **Email** — Any urgent unread messages? (Google credentials ✅ active)
- [x] **Calendar** — Upcoming events in next 24-48h? (Google credentials ✅ active)
- [ ] **Weather** — Relevant for Melbourne, AEST

## When to Reach Out

- Important email arrived
- Calendar event coming up (< 2h)
- Interesting finding from web search
- > 8h since last message

## When to Stay Quiet (HEARTBEAT_OK)

- Late night (23:00-08:00) unless urgent
- Clearly busy
- Nothing new since last check
- Checked < 30 minutes ago

## Memory Maintenance (Weekly)

- Review `memory/YYYY-MM-DD.md` files
- Distill to MEMORY.md
- Clean up outdated info
